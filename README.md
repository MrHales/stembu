# stembu: STellaris EMpire BUilder

A vibe coded collaboration between the creator, Mr.Hales, and Gemini inside Google Antigravity.

**stembu** is a Progressive Web App (PWA) designed to provide a comprehensive and deeply customizable empire builder for Stellaris, continuing the legacy of tools like MoMaker but tailored for a much deeper and richer galactic scale.

## Core Features (In Development)

The initial foundation includes placeholders and structure for:
- **Species Type & Traits**: Complete biological, lithoid, and machine trait support.
- **Homeworld & Home System**: Configure your deep space origin.
- **Name Lists & Flag**: Make your mark on the galaxy.
- **Origin & Government**: Design your ethical constraints and foundational origins.
- **Civics**: Robust civic dependencies that dictate the mechanics of your empire.

## Tech Stack
- Frontend: HTML/Vanilla JavaScript or React via Vite.
- Styling: Plain CSS emphasizing a deeply engaging, dark mode, glassmorphism aesthetic suitable for space-focused apps.
- Core: Installable PWA capabilities.

## Getting Started

1. Clone the repository.
2. Run `npm install`
3. Run `npm run dev` to start the local web development server.
